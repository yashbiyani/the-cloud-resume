## Introduction
It is a fully responsive personal portfolio website, responsive for all devices, built using HTML, CSS, and JavaScript.

## Prerequisites

Before you begin, ensure you have met the following requirements:

* [Git](https://git-scm.com/downloads "Download Git") must be installed on your operating system.

## Installation

Linux and macOS:

```bash
sudo git clone https://github.com/yashbiyani/Portfolio.git
```

Windows:

```bash
git clone https://github.com/yashbiyani/Portfolio.git
```
